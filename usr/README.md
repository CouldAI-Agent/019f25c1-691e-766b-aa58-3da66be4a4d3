# MPPSC Previous Year Questions

A Flutter application for browsing and practicing Madhya Pradesh Public Service Commission (MPPSC) previous year question papers.

## Features

- **Exam Categories**: Browse papers by category (Prelims and Mains).
- **Paper Listing**: View available question papers for selected categories and years.
- **Interactive Viewer**: Read questions with multiple-choice options.
- **Answer Reveal**: Practice by attempting questions and revealing the correct answer and explanation.
- **Cross-Platform**: Fully responsive design that works on mobile, tablet, desktop, and web.

## User Flows

1. **Dashboard**: Open the app to see exam categories.
2. **Select Category**: Tap on "Prelims" to view available general studies papers.
3. **Select Paper**: Choose a specific year's paper from the list.
4. **Practice**: Scroll through the questions, attempt them mentally, and tap "Show Answer" to verify and read the explanation.

## Tech Stack

- **Framework**: Flutter
- **UI Architecture**: Material 3
- **State Management**: Stateful widgets for local interaction state
- **Data**: In-memory mock data models (ready to connect to an API or Supabase)

## Setup and Run

Ensure you have Flutter installed.

```bash
# Get dependencies
flutter pub get

# Run the app
flutter run
```
